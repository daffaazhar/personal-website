HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

CSS

CSS (Cascading Style Sheet) is used to stylize elements written in a markup language. It separates the content from the visual representation of the site.

PHP

PHP is a server side scripting language that is embedded in HTML. It is used to manage dynamic content, databases, session tracking, etc.

JavaScript

JavaScript is a text-based programming language used both on the client-side and server-side that allow to make interactive web pages.

ReactJS

ReactJS is a library for building composable interface. It encourages the creation of reusable UI components, which present data that changes over time.

Sass

Sass (Syntactically awesome style sheets) is an extension of CSS that enables us to use things like nested rules, inline imports, and more.

Tailwind

A utility-first CSS framework packed with classes like pt-4 and text-center that can be composed to build any design, directly on the markup file.

NodeJS

Node.js is an open-source and cross-platform JavaScript runtime environment that runs on the V8 engine and executes JavaScript outside a browser.

WordPress

WordPress is a free and open-source website content management system (CMS) that is written in PHP and uses MySQL as the database
